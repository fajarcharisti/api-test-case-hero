# 🚀 AI Test Case Hero

Turn User Stories & API Contracts into structured QA test cases using AI (Claude) - instantly exportable to Excel.

## 🧠 What is this?

AI Test Case Hero is an AI-powered QA tool that generates API test cases from:

- 📖 User Stories  
- 🔗 API Contracts (OpenAPI / Swagger)  
- ⚡ Combined context (best coverage)

Then exports them into a ready-to-use Excel file for QA teams.

## ⚙️ Tech Stack

- Node.js (ESM)
- Anthropic Claude API
- ExcelJS
- Ora CLI Spinner
- Dotenv

## 🔄 Workflow

Story / API Contract → Claude AI Processing → Test Case Generation → Excel Export → QA Manual Review → API Test Case Ready for Automation

## ▶️ Run

``npm install``
``npm run generate``
